# glms

Visit: http://www.glms.zyx for more info and data

GLMS (Grapich Linux Monitoring System) je parce softvera koje sam poceo da pisem kao alternativa i prvenstveno za mene koje bi mozda zamenilo cacti i nagios.

Sastoji se iz:

- API
- GLMS client
- GLMS web server
- GLMS CLI

This project should be renamed, GLMS is just development name
